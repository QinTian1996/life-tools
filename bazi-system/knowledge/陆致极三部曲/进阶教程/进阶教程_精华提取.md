# 《八字命理学进阶教程》精华知识点提取

## 概述

《进阶教程》共13章，8028行，是在《基础教程》四视角分析框架（强弱、调候、格局、形象）基础上的**实务深化**。以下按专题提取与基础教程差异化的核心知识点。

---

## 一、强弱深化：度的量化（九级概念）

### 1.1 九级强弱体系

进阶教程在基础教程"极强→极弱"九级框架基础上，强调了**日主强弱的"度"**：

- 日主强弱不能决定命局好坏（曾国藩命造"很弱"却是中兴名臣）
- **日主只要不是太强或太弱即可**（谢武藤："调候第一、格局第二、日主第三"）
- 核心在于日主**是否有根**，而非简单的计分结果

### 1.2 根的系统分类（第2章B节）

**"根"的定义**：地支中出现的同类五行成分（主气或附属之气）

**通根地支表**（阳干木火金水）：

| 五行 | 长生 | 临官（祿） | 帝旺（刃） | 衰 | 墓 |
|------|------|-----------|-----------|-----|-----|
| 木   | 亥   | 寅        | 卯        | 辰  | 未  |
| 火   | 寅   | 巳        | 午        | 未  | 戌  |
| 金   | 巳   | 申        | 酉        | 戌  | 丑  |
| 水   | 申   | 亥        | 子        | 丑  | 辰  |

- **强根**：临官（祿）、帝旺（阳刃）
- **次强**：衰位
- **弱根**：长生、墓（两头位置）

### 1.3 阴阳生死差别（关键深化）

**"阳生阴死、阴生阳死"论**——《子平真诠》主张阳干为"气"、阴干为"质"：

- 阳干之"死"≠阴干之"死"
  - 甲木死午→耗泄其气，真死
  - 乙木死亥→亥中壬甲助身，假死
- 阳干之"绝"——绝地逢生（甲绝申，申中庚生壬水生甲）
- 阴干之"绝"——真绝（乙绝酉，酉金直砍乙木）

### 1.4 根的强度判断法

观察天干在地支的"根"的状况，可初步判断强弱：
- 有根：可论身强（需配合月令判断）
- 无根：日主再旺也有限
- **印星≠根**——这是核心突破

## 二、根与印的本质区别（第2章C-D节）

### 2.1 根本性差异

| 对比项 | 根（比劫） | 印星 |
|--------|-----------|------|
| 关系 | 同我（兄弟姐妹） | 生我（父母长辈） |
| 作用 | 日主立足之本 | 生助日主 |
| 替代性 | 不可替代 | 不能替代根 |
| 过量后果 | — | 母多灭子 |

### 2.2 "母多灭子"机制（第2章C节）

**核心原理**：印星旺 ≠ 日主强

《滴天髓》"反局"——印星太旺反埋没日主：
- 典型案例：辛金日主，土重重（四柱皆土）→ "土重金埋，母多灭子"
- 实例：戊辰 丙辰 辛丑 戊辰
- 原因：日主自身弱、地支无强根、干头无比劫
- 应断：喜用反为扶日主、制印星，而非跟着"母"走

**同类现象**：
- 水旺木漂
- 木旺火塞
- 火多土焦
- 土多金埋

### 2.3 印星≠根的实战意义

- 无根的日主**旺不了**（印星再多也没用）
- 有根+印星→旺度增强
- 无根+无印→极弱
- **官杀两头蛇**：官杀本克身，但若天干透印，官杀生印、印再生身 → 官杀反起扶身作用（"贪生忘克"）

## 三、调候深化：计分细化与配置优化（第2章）

### 3.1 寒暖燥湿计分回顾

天干计分：甲+3、乙+1、丙+6、丁+4、戊+5、己-4、庚-1、辛-3、壬-5、癸-6
地支计分：寅+3、卯+1、辰-4、巳+5、午+6、未+3、申-2、酉-3、戌+4、亥-5、子-6、丑-4
月令加分：寅0、卯+1、辰+2、巳+3、午+4、未+3、申0、酉-1、戌-2、亥-3、子-4、丑-4

**中和范围**：-6 ~ +6（超出即偏颇）

### 3.2 实战突破：调候不足≠命局废

**章太炎命例核心观点**：
- 寒暖燥湿度=-14（严重寒湿），全局无火
- 但正官格，官印相生，"贵而不富"
- **大运可以补足调候缺陷**——他中年走南方火地而腾飞
- **结论**：不能因调候不足就一棍打死，须看大运能否补充

### 3.3 天干优化配置（《穷通宝鉴》精华）

十日干最优组合 → 核心突破在基础之上：

| 日干 | 最优配置 | 含义 |
|------|---------|------|
| 甲 | 丁、庚 | 用庚劈甲，方引丁火——**甲丁庚循环** |
| 乙 | 丙、癸 | 花草喜阳光雨露 |
| 丙 | 壬 | 日照江湖，水辅阳光 |
| 丁 | 甲、庚 | 引火须甲，炼金须庚 |
| 戊 | 甲、丙、癸 | 疏土+照暖+润泽 → 土最复杂 |
| 己 | 甲、丙、癸 | 同上 |
| 庚 | 丁、甲 | 火炼成器，甲木为原料 |
| 辛 | 壬、甲 | **辛喜壬淘洗，非庚喜火炼** → 重大区别 |
| 壬 | 庚(辛) | 水源远流长 |
| 癸 | 辛(庚) | "金白水清" |

**甲丁庚循环**：对甲，调候用神为庚丁；对丁，调候用神为甲庚；对庚，调候用神为丁甲——三者循环为用，是命理学一条重要准验理论。

**辛金特性**：与庚金喜丁火锻炼完全不同，辛金喜壬水淘洗（珠宝首饰需水冲刷）。辛有"乐水之盈"的特点，怕土来制水（土厚埋金），故需甲木破土。

## 四、格局深化：变通法则与六条评判标准

### 4.1 格局变通法则

**（1）数量变性质**：某一成分多过3个以上（主气），"善神"变"凶神"
- 官多变杀（例：癸卯 甲寅 己亥 丙寅）
- 食多变伤（例：章士钊——癸水卯月，乙食神众多）
- 正印变枭神
- **仍然符合"善顺用、凶逆用"法则，但要变通**

**（2）格局不显时移日支取格**
- 月令不透、日支透出食神生财→取日支为格
- 例：乙酉 戊子 甲寅 丙子——日支寅为枢纽

### 4.2 格局高低六条评判标准

进阶教程系统化提出格局高低的评判维度：

| 序号 | 标准 | 解释 |
|------|------|------|
| 1 | **真假** | 格局是否得月令真神（真者有力，假者虚浮） |
| 2 | **虚实** | 干支虚实关系（天干有根有生者实，无根无生者虚） |
| 3 | **清浊** | 清纯不杂（如官杀不混）vs 混杂破格 |
| 4 | **有力/无力** | 各成分是否通根强健、天覆地载 |
| 5 | **有情/无情** | 各成分配合是否得宜，用神是否完好不受制 |
| 6 | **团结** | 全局是否和谐一体，能否形成统一的力量方向 |

### 4.3 干支虚实表（盲派引入）

| 天干 | 实 | 虚 |
|------|----|----|
| 甲 | 寅、辰、子 | 申、戌、午 |
| 乙 | 亥、卯、未 | 巳、酉、丑 |
| 丙 | 寅、午、戌 | 子、申、辰 |
| 丁 | 巳、卯、未 | 亥、丑、酉 |
| 戊 | 戌、午、辰 | 子、申、寅 |
| 己 | 巳、未、丑 | 亥、酉、卯 |
| 庚 | 申、辰 | 子、午、寅、戌 |
| 辛 | 丑、酉 | 巳、亥、未、卯 |
| 壬 | 申、子、辰 | 戌、午、寅 |
| 癸 | 亥、酉、丑 | 巳、未、卯 |

**实战意义**：虚的财星不表示财富，而表示才能（胡适例）。

## 五、刑冲会合深入（第3章）

### 5.1 六合新解（盲派视角）

**六合分为两类**：
- 剋合：子丑（土剋水）、卯戌（木剋土）、巳申（火剋金）
- 生合/助合：寅亥（水生木）、辰酉（土生金）、午未（火生土）

**合的作用**：不只讲合化，更关注因合产生的内部生剋、动静脉——妻宫被合会成"妻子与人跑"（寅亥合例）。

### 5.2 六冲新论

**"盲派"**：相冲本意是往来互换，表示得到、互有、交换、保持联系。只有冲中带剋才有破坏作用。

**紧冲 vs 遥冲**（《滴天髓》）：
- **紧冲为剋**（相邻）
- **遥冲为动**（隔位）

**冲忌神反为有利**：辰戌冲若拔去从财格中的微根→格局澄清。

### 5.3 六害（穿）——盲派特别重视

| 六害 | 程度 |
|------|------|
| 子未穿 | 相剋带穿，最严重 |
| 卯辰穿 | 相剋带穿，最严重 |
| 酉戌穿 | 相剋带穿，最严重 |
| 丑午穿 | 次之 |
| 寅巳穿 | 次之 |
| 亥申穿 | 次之 |

**穿的含义**：排斥、不容、破坏、伤害。正的可以被穿偏，偏的可被穿正（"象"的思维）。

### 5.4 暗合（第3章H节）

- 寅丑暗合
- 午亥暗合
- 卯申暗合

### 5.5 地支相破（盲师特有）

诗诀所载的破的关系（子卯破、丑寅破、卯午破、亥辰破、申未破、午酉破、戌巳破），用于类象推演。

## 六、拱夹法则与虚神（第2章）

### 6.1 暗拱原理

- 亥+未（缺卯）→ 暗拱卯木（虚神）
- "向实寻虚，从无取有"
- 一旦天干透出甲/乙→"邀进"卯木虚神→构成完整三合局

### 6.2 拱的实战意义

曾国藩命例核心：年支未+月支亥暗拱卯木。46岁甲午运甲木透出→邀进虚神卯→亥卯未三合木局→印星助身→风云际会

### 6.3 生旺库会合期（第4章）

十二地支分三组：四生（寅申巳亥）、四旺（子午卯酉）、四库（辰戌丑未）

**会合期规则**：当命局+大运+流年凑齐某组全部四个地支→当年必应凶事

- 例1：命局辰戌、大运未、流年丑→四库会合期→妻宫外孕死亡
- 例2：命局寅申、大运亥、流年巳→四生会合期→家业付之一炬

## 七、六亲分析：星与宫双线（第5-8章）

### 7.1 六亲星宫体系

**男命**：正印=母，偏财=父，正财=妻，七杀=子，正官=女，比劫=兄弟
**女命**：正印=母，偏财=父，正官=夫，伤官=子，食神=女，比劫=兄弟

**宫位**：年=祖上，月=父母兄弟，日干=己身/日支=配偶，时=子女

### 7.2 星宫双线分析

1. **星自身旺衰**：月令→通根→原神→刑冲会合影响
2. **星与宫关系**：星在本宫=安分；星游走他宫=可能迁移/分离
3. **星与日主亲疏**（位置编号法）：日支1、月干2、时干3、月支4、年干5、时支6、年支7
4. **星对日主的喜忌**：用神/忌神

### 7.3 早年丧父母的核心规律

- 父母星弱，怕会合刑冲剋害
- 父星/母星透干遇剋→"比劫重重定剋父""财多而母年不固"
- 三合局合去藏干母星/父星→危险

### 7.4 妻宫动静看婚期

**核心逻辑**：婚期=大运/流年引起夫妻宫状态发生变化的时刻
- 妻宫由静变动（冲、合、刑）→婚期信号
- **印星+配偶星同时感应**→领证年（印为文书）

### 7.5 女命八法现代诠释

**古法四吉**：纯、和、清、贵 → 官星一位、有力、有情、不被伤
**古法四凶**：濁、濫、娼、淫 → 官杀混杂、伤食太重、比劫旺

**现代突破**：时代变了，女性可自主婚姻、职业发展，旧法需调整

## 八、财官网络三大领域（第10-11章）

### 8.1 经济领域（财星）

**"财气通门户"真义**：财星从月令（门戶）中透出→真神得用→质量重于数量

**发富模式**：
- 财气通门户（最优）
- 伤食生财（次优）
- 身弱财旺→"富屋贫人"
- 财宜藏不宜露→"枪打出头鸟"

**比劫与财的对立**：
- 比劫夺财→需伤食调停（比劫→伤食→财）
- 或官杀制比劫护财

### 8.2 政治领域（官杀）

**五大贵格形态**（进阶教程系统化）：
1. 官印相生——正官格身弱，文贵
2. 财官双美——正官格身强
3. 杀印相生——七杀格身弱，文武双全
4. 财滋弱杀——七杀格身强
5. 食神制杀——身杀两停，武贵
6. 伤官驾杀/阳刃驾杀——变体

**伤官佩印取贵**：伤官用印，见官杀运时得权贵地位

### 8.3 知识领域（印+食伤）

- 印星=吸收（学历、教育程度）
- 食伤=释放（才华、创造力）
- **正印→传统学术；偏印→非传统/技术**
- **食神→内向含蓄；伤官→外向多才**

## 九、岁运深入实战要点（第4章、第12章）

### 9.1 第六步大运"死关"

**核心原理**：第六步大运总是"冲提"大运（与月柱天剋地冲）

- 1岁起运者→51~60岁冲提
- 10岁起运者→60~69岁冲提
- 人人70岁前必走第六步冲提运
- 若喜用在月柱→直接受冲击→再遇不吉利流年→寿元尽头
- 现代医疗改善，人均寿命延长，但冲提依然是大关

### 9.2 连环式刑冲

**四种致命组合**：

1. **反吟伏吟连环发生**
   - 命局内反吟+岁运反吟/伏吟串联
   - 例：月柱辛酉+日柱乙卯（反吟），大运乙丑+流年辛未（再反吟）→车毁人亡

2. **冲合阳刃**
   - 阳刃为忌+逢冲合→"阳刃冲合岁君，悖然祸至"

3. **三冲一**
   - 身弱逢三冲一→根拔→性命之忧
   - 例：杨宇霆——三辰冲一戌→死于非命

4. **方局相混**
   - 三会方+三合局同时出现→力量过度强旺
   - 例：徐树铮——申子辰局+亥子丑方→水盛木漂

### 9.3 晦气流年与日犯岁君

- **流年太岁剋日主**：有灾不为大害
- **日主剋流年太岁**（"日犯岁君"）：可能大灾
  - 若天干有一字通关→反凶为吉，有进财机会

### 9.4 大运祿冲提

**祿运的危机**：祿是日主在地支的延伸，代表日主身体
- 曾国藩：癸巳运（丙祿）→双冲月时亥水→脑溢血
- 杜月笙：丙寅运冲庚申→"冲提"→格局破→辛卯年身故

### 9.5 换甲寿关

- 不同方位大运交脱（如北方→东方）→"换甲"
- 老年气衰→不堪剧烈变化→死在运底
- "换得過是人，换不過是鬼"

### 9.6 大运看吉凶、流年看应期

盲派核心方法论：
- 大运提示事件范围（如官杀运=婚姻运）
- 流年触发应期（岁运并临、伏吟、合冲等）
- 大运重地支方向（东南西北中）
- 流年为君、大运为臣

## 十、寿元与灾祸（第12章）

### 10.1 寿元核心

《滴天髓》："何知其人寿？性定元氣厚"

- **食神为寿星**：食神通根有力→长寿
- **五行元氣皆厚，四柱通根生旺，左右上下有情**→极贵长寿
- **日主无气**→忌走衰败死绝之地

### 10.2 寿关通则

- 格局不可破（正官格不走伤官运，印格不走财运，财格不走比劫）
- 用神及其原神**同时被剋破**→大难临头
- 日主之祿逢刑冲剋害→性命攸关

### 10.3 官非牢狱信号

- 伤官见官（伤官格走官运）
- 七杀猖狂、官杀混杂
- **三刑一/一刑三**
- **天罗地网**（辰巳+戌亥）
- **多重魁罡**
- **用神遭刑**（被刑者为喜用神）

### 10.4 五行偏颇致凶

- 火炎土焦、土多金埋、水旺木漂等→不得善终
- "气浊神枯"→日主失令，用神浅薄，忌神深重

## 十一、疾病分析（第13章）

### 11.1 双线分析

1. **全局五行分布**→各脏腑能量分布→五行偏颇→潜在疾病因子
2. **从日主出发**→生剋制化→对日主的影响方式

### 11.2 关键法则

- **从弱脏下手**：五行中最弱的一方是疾病切入点
- **星+宫位交叉**：如庚金在月柱（胸部）=肺
- **"星"弱→疾病因子**：丁火弱=心脏弱，庚金遭剋=肺癌
