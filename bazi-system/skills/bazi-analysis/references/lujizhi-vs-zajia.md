# 陆致极 vs 杂家古籍 — 实战工具箱对照

> Generated from session 2026-05-08 after re-learning Lu Jizhi trilogy

## 核心定位

| 体系 | 性质 | 适合用途 |
|------|------|---------|
| **陆致极** | 现代系统化教科书 | 框架定调、量化分析、理论理解 |
| **杂家古籍** | 传统经验实战手册 | 实战经验、应期判断、细节抠查 |

## 各环节最佳选择

| 分析环节 | 用陆致极的 | 用杂家的 | 优先级 |
|---------|-----------|---------|:------:|
| **排盘定旺衰** | 九级强弱系统 + 三步法 | 韦千里论身强/论身弱（模糊） | 🥇陆 |
| **找用神** | 三类用神分开找 + 三用合一 | 子平真诠格局用神专求月令 | 🥇陆 |
| **调候判断** | 计分系统（±6阈值） | 穷通宝鉴120组调候表 | 🥇陆 |
| **格局定贵贱** | 六条评判（真假/虚实/清浊等） | 子平真诠格局成败救应 | 🥇持平 |
| **大运流年吉凶** | 拱夹/反吟伏吟框架 | 征太岁/十干败亡诗/应期法 | 🥇杂 |
| **具体应期/月份** | 理论框架较薄弱 | 三刑应期/六冲应期/伏吟应期 | 🥇杂 |
| **六亲分析** | 星宫双线框架 | 星家十要 + 六亲细化 | 🥇杂 |
| **心理/哲学层面** | 黑箱理论/符号学/丙火照己 | 无 | 🥇陆 |
| **神煞** | 基本不用 | 精选3-5个（天乙/华盖/驿马/桃花/阳刃） | 🥇杂* |
| **纳音** | 完全不用 | 辅助参考 | 🥇杂* |
| **日常决策** | 无 | 五行宜忌完整体系 | 🥇杂 |

> *注：神煞纳音按用户偏好保留使用，但判断"没用"时需单独说明原因。

## 实战流程

```
Step 1 — 陆致极框架定调
├── 排八字大运流年（确认节气无误）
├── 强弱分析（九级定位：得令/得地/得助）
├── 调候分析（计分系统看寒暖变化）
├── 格局分析（月令透干取格 + 六条评判）
├── 形象检查（有无特殊气势）
├── 三用合一 -> 确定核心用神

Step 2 — 杂家经验细化
├── 征太岁检查（流年干支是否冲刑日柱）
├── 十干败亡诗检查（弱日主是否走到绝地）
├── 行运判断法（各干支的吉凶表现）
├── 神煞参考（天乙/华盖/驿马等）
├── 纳音参考（辅助判断）
├── 具体应期（哪个月份容易出事/好事）

Step 3 — 综合结论
├── 整体评价（吉/凶/平/吉凶参半）
├── 具体建议（注意什么、把握什么）
├── 丙火照己视角下的心法总结
```

## 当两者冲突时怎么办

1. **框架优先**：结构定调以陆致极为主（四个视角的系统性碾压单一视角）
2. **细节取杂**：应期、六亲、具体事象以杂家经验为准（陆致极在这些方面偏薄弱）
3. **三用合一是最高准则**：如果三种用神指向同一个五行，那就是铁板钉钉
4. **神煞不破格**：神煞可做参考，但不推翻格局用神的核心判断
