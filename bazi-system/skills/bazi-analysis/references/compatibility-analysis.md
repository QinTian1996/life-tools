# 合婚分析 (Compatibility Analysis) — Methodology Notes

Based on session analysis of three different Bazi pairs, May 2026.

---

## Core Framework

| Dimension | Analysis | Weight |
|:---:|:---|:---:|
| **日干关系** | 五行生克 (生/克/比和) | High — 天干关系决定相处基调 |
| **日柱关系** | 刑冲合害 (especially 夫妻宫) | **Highest** — 日支冲=大忌 |
| **年柱关系** | 刑冲合害 | Medium — 根基是否相合 |
| **月柱关系** | 刑冲合害 + 五行 | Medium — 内在需求是否一致 |
| **五行互补** | 双方喜忌是否互补 | Medium-High |
| **通关可能** | 能否形成流通链条 | Medium |

## 通关 (Flow/Conduction) Analysis

The most important subtlety: a 克 relationship can be transformed if a **中介 (mediator)** exists.

Example from session:
- 男命土旺(戊), 女命金旺(辛)
- 土克水 (表面上是克)
- 但土 → 金 → 水 (土生金, 金生水)
- 所以男命之土通过女命之金, 转化为生助女命之水的能量
- **结论: 有通关, 克而不害**

Formula: 克方 → 中介 → 被克方 = 通关

## Common Pitfalls

1. **日支冲 (辰戌冲, 子午冲等)** — 夫妻宫对冲是硬伤, 即使其他方面好也需要极用心经营
2. **官杀混杂 × 伤官** — 女命官杀混杂+男命伤官, 感情中易互相挑剔
3. **比劫夺财** — 男命比劫重+女命财星重, 经济上易有矛盾
4. **印星过重** — 双方印都重, 容易缺少激情
