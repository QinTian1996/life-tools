**prompt 管推理流程，rules 存参考数据，不把全文知识库塞进上下文。**

### SKILL.md — 主控制逻辑（17 KB）

从 bazi-analysis skill 完整提取。包含：

- **陆致极四视角框架**：强弱 / 调候 / 格局 / 形象
- **三用合一判定法**：扶抑用神 / 调候用神 / 格局用神 是否重合
- **调候计分速查表**：天干计分 + 地支计分 + 月令加分
- **韦千里八步法**：强弱→格局→用神→喜忌→岁运→六亲→性情→事业
- **起运计算**：数节不数气的正算法 + 已验证案例
- **十干败亡诗**：身弱七杀忌走绝地
- **神煞速查**：天乙贵人 / 华盖 / 驿马 / 桃花 / 阳刃 / 将星 的查法口诀
- **输出格式规范**：专业版 + 详解版 双报告要求
- **PDF 生成流水线**：pandoc + chromium headless 命令
- **常见错误**：起运算错（中气 vs 节）、性别规则、腊月陷阱

## 使用方式

### 注入 prompt

将 `SKILL.md` 作为 system prompt，模型即可独立完成排盘到报告的完整流程。无需外挂知识库。

### 参考数据按需检索

`rules/三部曲核心框架.md` 可以作为 RAG 数据源，在需要查阅具体规则时检索对应段落。

### 生成 PDF

```bash
pandoc 分析报告.md \
  --template=templates/report-template.html \
  --metadata title="命理分析报告" \
  -o 分析报告.html --self-contained

chromium --headless --disable-gpu --no-sandbox \
  --print-to-pdf=分析报告.pdf \
  file:///path/to/分析报告.html
```

## Token 预估

- 17 KB prompt + 按需检索 5-15K = **一次完整分析约 25-40K tokens**
- 只出单报告：**20-30K tokens**
- 最简分析（排盘+用神+一句话总结）：**15-20K tokens**

## 能力边界

| ✅ 能做 | ❌ 不能做 |
|:---|:---|
| 排盘（公历→八字） | 引用古籍原文佐证 |
| 强弱判断 + 用神取定 | 调取陆致极具体章节逐句论证 |
| 大运排定 + 流年分析 | 调取陆致极图片辅助讲解 |
| 双报告输出（专业版+详解版） | 合婚详细分析（需加载 reference） |
| PDF 生成 | 紫微斗数 / 六爻等跨领域分析 |
| 神煞分析（核心6个） | 190+ 神煞全量铺开 |
