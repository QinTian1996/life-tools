### T002: Write Typography Token Specification

<action>
Write the typography section in `.planning/design-tokens.md`.

**Font Families:**
| Token | Font | Fallback | Usage |
|-------|------|----------|-------|
| font-brand | LxgwWenKai Mono | system-ui, sans-serif | 品牌名、标题、轻松文本 |
| font-mono | Sarasa Mono SC | ui-monospace, monospace | 代码、技术内容 |

**Installation:**
- LxgwWenKai: `npm install @fontsource/lxgw-wenkai` then import in layout.tsx
- Sarasa Mono SC: Download TTF from GitHub releases, use `next/font/local`

**Font Loading Config (Next.js 16):**
```typescript
// app/fonts.ts
import localFont from "next/font/local";

export const sarasaMono = localFont({
  src: "./fonts/SarasaMonoSC-Regular.ttf",
  variable: "--font-sarasa-mono",
  display: "swap",
  preload: false,
  fallback: ["ui-monospace", "monospace"],
});
```
