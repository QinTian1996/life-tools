### Error Code Mapping (D-01, D-03)
```
if message.includes('rate') → RATE_LIMITED
if message.includes('context') || message.includes('token') → CONTEXT_OVERFLOW
if message.includes('invalid') → INVALID_REQUEST
else → MODEL_ERROR
```

- messages: required, non-empty array
- Each message: { role: string, content: string }
- temperature: optional number 0-2

```bash
  -d '{}'

# 5. Error: empty messages

  -H "Content-Type: application/json" \

# Expected: 400 {"error":"messages must be a non-empty array","code":"INVALID_REQUEST"}
```
